🛠️ Solução de Problemas com API do Google Maps

Se o erro for sobre **Referer/Restrição** ou **REQUEST_DENIED**, verifique os seguintes 4 pontos no seu **Google Cloud Console**:

  1. Chave (API Key): Certifique-se de que a chave inserida no código é exatamente a mesma que você gerou.
  2. APIs Ativadas: Confirme que as seguintes APIs estão **ativadas** no seu projeto:
    • ✅ **Geocoding API**
    • ✅ **Directions API**
  3. Faturamento (Billing): O faturamento (cartão de crédito) precisa estar **ativado** no projeto. O Google só cobra se você exceder o crédito mensal de \$200 USD.
  4. 🔴 Restrições de Chave (MAIS PROVÁVEL): Se a chave tem restrição de Referer HTTP, ela não funciona em ambientes sandbox. Para testes, mude a restrição de chave para **Nenhuma** (sem restrição) ou restrinja apenas por API (item 2).
    **Se for usar restrição HTTP Referer, adicione `*` (asterisco) para permitir qualquer URL temporariamente.**
